# mel

A crossplatform OpenGL game framework written in C with focus on stability and speed.

## Features

- Crossplatform
- Logging
- Shaders
- 2D Batching
- Texture Rendering
- Rect Rendering
- Camera Support
- Different transformations (scaling, rotating, ...)
- Multithreading

## Installation

Just clone/download the repository and you're good to go.
